# 命名规范

本文档定义文件命名和变量命名的规则。

## 文件命名

| 类型 | 规范 | 示例 |
| ---- | ---------- | ------------------- |
| 组件文件 | PascalCase | `InstanceList.tsx` |
| Hook 文件 | camelCase | `useAuth.ts` |
| 工具函数 | camelCase | `formatDate.ts` |
| 类型文件 | camelCase | `index.ts` |
| 样式文件 | kebab-case | `global-styles.css` |

## 变量命名

```tsx
// 组件名：PascalCase
function InstanceList() {}

// 函数名：camelCase，动词开头
const fetchData = () => {};
const handleClick = () => {};
const formatDate = () => {};

// 布尔值：is/has/can 前缀
const isLoading = true;
const hasError = false;
const canEdit = true;

// 常量：UPPER_SNAKE_CASE
const API_BASE_URL = 'http://...';
const MAX_RETRY_COUNT = 3;

// 事件处理：handle 前缀
const handleSubmit = () => {};
const handleChange = () => {};

// 回调属性：on 前缀
interface Props {
  onSuccess?: () => void;
  onChange?: (value: string) => void;
}
```

## 函数命名动词

| 动词 | 用途 | 示例 |
| ---- | -------- | ----------------- |
| get | 获取数据 | `getInstance()` |
| set | 设置数据 | `setInstance()` |
| fetch | 异步获取 | `fetchData()` |
| handle | 事件处理 | `handleClick()` |
| is/has/can | 布尔判断 | `isValid()` |
| create | 创建 | `createInstance()` |
| update | 更新 | `updateInstance()` |
| delete | 删除 | `deleteInstance()` |
| format | 格式化 | `formatDate()` |
| parse | 解析 | `parseJSON()` |
| validate | 验证 | `validateEmail()` |

## Hook 命名

```tsx
// 自定义 Hook 以 use 开头
function useAuth() {}
function useLocalStorage() {}
function useDebounce() {}
function useInstances() {}
```

## 组件 Props 命名

```tsx
interface Props {
  // 数据属性：名词
  instance: Instance;
  items: Item[];

  // 布尔属性：is/has/can 或形容词
  isOpen: boolean;
  disabled: boolean;

  // 回调属性：on + 事件名
  onClick: () => void;
  onChange: (value: string) => void;
  onSubmit: (data: FormData) => void;

  // 可选属性带 ?
  className?: string;
  style?: React.CSSProperties;
}
```

## CSS 类名

```tsx
// 使用 kebab-case
<div className="instance-list">
  <div className="instance-item">
  <div className="instance-info">
</div>

// 配合 Ant Design 使用时
<div className="flex items-center gap-4">
```

## 文件夹命名

```
components/         # 小写
  InstanceList/      # PascalCase（组件文件夹）
    index.tsx
    InstanceList.tsx
pages/              # 小写
hooks/              # 小写
services/           # 小写
```
